About This Project

This is a sleek, real-time currency converter built with modern web technologies including HTML, CSS (Tailwind), and vanilla JavaScript. It leverages the reliable Frankfurter API to provide accurate and up-to-date financial data. The project showcases dynamic UI elements, scroll-triggered animations, and a fully responsive design for a seamless user experience.
