A Frontend Playground

This dashboard is a hands-on project designed to showcase modern frontend development techniques. It's built with vanilla HTML, CSS, and JavaScript, and it interacts directly with the Reqres.in mock API. The goal is to demonstrate how to build a dynamic, responsive, and visually appealing web application without relying on any frameworks.

Key Features Demonstrated:

  • ✔️ API Interaction: Fetching, creating, and displaying data from a live REST API.
  • ✔️ Dynamic UI: Rendering data on the fly and updating the page without a reload.
  • ✔️ Advanced CSS: A dark-mode theme with gradient effects, custom animations, and a responsive layout.
  • ✔️ User Experience: Interactive elements like modals, loaders, and smooth, scroll-triggered animations.
