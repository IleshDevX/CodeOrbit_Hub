About Economic Data Visualizer

EcoViz is a modern web application that lets you explore, search, and visualize thousands of economic data series from the FRED API (Federal Reserve Economic Data).
Built for students, researchers, and anyone interested in economic trends, EcoViz provides interactive charts, rich metadata, and a beautiful user experience.
All data is sourced live from the St. Louis Fed and updated regularly.
